## 🎯 Sobre

Este repositório contém materiais de estudo para o exame de Certificação GitHub Foundations. O conteúdo está estruturado de acordo com o [guia oficial de estudos](https://assets.ctfassets.net/wfutmusr1t3h/1kmMx7AwI4qH8yIZgOmQlP/79e6ff1dfdee589d84a24dd763b1eef7/github-foundations-exam-study-guide__1_.pdf) e segue a iniciativa da Comunidade [GitHub Foundations Exam Prep](https://github.com/orgs/community/discussions/154502#discussion-8110303).

- [Português](README.md)

## 📚 Domínios de Estudo

A certificação abrange 7 domínios principais:

| Domínio | Peso | Tópicos |
|---------|------|---------|
| [Fundamentos do Git e GitHub](foundations_of_foundations/git_and_github_basics.md) | 22% | Controle de versão, fundamentos do Git, recursos do GitHub |
| [Trabalhando com Repositórios GitHub](repositories/working_with_gh_repositories.md) | 8% | Gerenciamento de repositório, colaboração |
| [Recursos de Colaboração](repositories/collaboration_features.md) | 30% | Pull requests, issues, discussões |
| [Desenvolvimento Moderno](copilot/modern_development.md) | 13% | GitHub Copilot, Actions, Codespaces |
| [Gerenciamento de Projetos](github_features_and_open_source/project_management.md) | 7% | Projects, quadros, acompanhamento de trabalho |
| [Privacidade, Segurança e Administração](enterprise_essentials/privacy_security_administration.md) | 10% | Controle de acesso, recursos de segurança |
| [Benefícios da Comunidade GitHub](github_features_and_open_source/benefits_of_the_gh_community.md) | 10% | Código aberto, engajamento com a comunidade |

## 🎓 Roteiro de Estudos

1. Comece com os [Fundamentos](foundations_of_foundations/README.md) para aprender Git e GitHub básico
2. Avance para [Desenvolvimento Moderno](copilot/README.md) para entender as ferramentas de desenvolvimento do GitHub
3. Aprenda sobre [Gerenciamento de Repositórios](repositories/README.md)
4. Explore os [Recursos do GitHub](github_features_and_open_source/README.md) e aspectos da comunidade
5. Complete com [Essenciais para Empresas](enterprise_essentials/README.md)
